# Atom Tmp Memo

An Atom package that add memo space.

## Motivation

- I wanted a little space that was not affected by the text editor part.
- I wanted to keep copied pasted ones.

## Install

```
apm install tmp-memo
```

## Usage

- Run `Tmp Memo: Toggle` command.
- Use `ctrl-alt-0`
